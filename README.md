## 💼 Featured Project: Mentorship Platform (Back-End System)

A full backend for a web platform connecting mentees with expert mentors, built with a focus on clean API design and real-world business logic.

**What I built:**
- 🔐 JWT-based authentication and role-based access control (RBAC) for mentors, mentees, and admins
- 📅 Scheduling and booking system with conflict detection, integrated with the Google Calendar API
- 🧩 RESTful CRUD APIs for user profiles, mentor-mentee data, and session management
- 🛡️ Admin-only endpoints for platform moderation and user management
- 🤝 Built as part of a 2-person backend team, with shared ownership across auth, scheduling, and admin modules

🛠 **Tech Stack:** Node.js, Express.js, TypeScript, Prisma ORM, MySQL

🔗 **Repository:** https://github.com/AhmadAlBarasy/mentorship-platform-api/
