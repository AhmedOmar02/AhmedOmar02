## 🤖 Other Project: BetweenTheLines.AI (AI-Powered Full-Stack App)
A full-stack web application that analyzes a sentence and generates multiple possible interpretations, helping users avoid miscommunication before they respond.

**What I built:**
- 🧠 AI integration with DeepSeek via the OpenAI-compatible SDK to generate 2–4 distinct interpretations per sentence
- 🎭 Tone detection per interpretation (neutral, frustrated, sarcastic, anxious, affectionate, etc.) and a dominant tone summary
- 📝 Optional context fields (relationship type, channel, background) to ground interpretations in the user's situation
- ⚡ Stateless REST API with clean separation: AI logic isolated in a dedicated service layer, keeping the provider swappable
- 🖥️ React frontend for submitting sentences and displaying interpretations with their explanations and tones

🛠 **Tech Stack:** Node.js, Express.js, React, DeepSeek API
